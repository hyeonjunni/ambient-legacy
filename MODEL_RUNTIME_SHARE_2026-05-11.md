# 로컬 모델 비교 요약 (톡방 공유용)

## 실험 조건

- 기준: 가족 기록 기반 RAG형 질의응답
- 런타임: Ollama 로컬 실행
- 환경: Apple Silicon 맥북 18GB
- 페르소나: `father-calm`
- 질문 3종:
  - 송년회 발화 2문장 요약
  - 김치찌개 메모 3단계 정리
  - 기록에 없는 부산 여행 정확한 날짜/식당 이름 거절

## 한눈에 결론

- 현재 데모 기본값으로는 `Gemma 4 E2B`가 가장 무난했다.
- 같은 Gemma 계열 상위선인 `Gemma 4 E4B`는 더 보수적인 거절을 보였지만, 응답 시간이 크게 늘었다.
- 8B급 확장 후보는 두 갈래로 보인다.
  - 속도-품질 균형: `EXAONE 3.5 7.8B`
  - 거절 안정성/정돈: `Qwen 3 8B`

## 추천 정리

- 발표/데모 기본값:
  - `Gemma 4 E2B`
- 같은 Gemma 계열 상위 비교 후보:
  - `Gemma 4 E4B`
- 8B급에서 속도 쪽을 먼저 볼 때:
  - `EXAONE 3.5 7.8B`
- 8B급에서 보수적 거절과 정돈을 볼 때:
  - `Qwen 3 8B`

## 해석 시 주의

- 이번 결과는 `더미 가족 기록 + 동일 persona + 동일 retrieval 포맷` 기준이다.
- 따라서 절대 성능 비교라기보다, 현재 Ambient Legacy 데모 구조에서의 응답 경향 비교로 보는 편이 맞다.
